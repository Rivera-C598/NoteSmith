# NoteSmith - Smart Note-Taking Application

## For Students: Quick Start Guide

### System Requirements
- **Operating System**: Windows 10 or later
- **No Java installation needed!** (Java is bundled)

### Installation Steps

1. **Extract the ZIP file**
   - Right-click `NoteSmith-Portable.zip`
   - Select "Extract All..."
   - Choose a location (e.g., Desktop, Documents)
   - Click "Extract"

2. **Run NoteSmith**
   - Open the extracted `NoteSmith` folder
   - Double-click `NoteSmith.exe`
   - The app will start immediately!

3. **First Time Use**
   - Click "Register" to create your account
   - Choose a username and password (min 8 characters)
   - Start taking notes!

### Features

✅ Create and organize notes  
✅ Add tags to categorize notes  
✅ Pin important notes  
✅ Search across all notes  
✅ Export to Markdown or HTML  
✅ Optional AI features (requires free API key)  
✅ Secure password protection  
✅ Automatic save - never lose your work!  

### AI Features (Optional)

To enable AI-powered features like smart summaries and note linking:

1. Click the **Settings** button (top right)
2. Click **"How to Get Free API Key"**
3. Follow the instructions to get a free Google Gemini API key
4. Paste your key and click Save
5. Restart the app

**Note**: AI is completely optional - the app works great without it!

### Your Data

- All your notes are stored locally on your computer
- Location: `C:\Users\YourName\.notesmith\`
- Your data is private and stays on your computer
- No internet connection required (except for optional AI features)

### Troubleshooting

**App won't start?**
- Make sure you extracted the entire ZIP file
- Don't move `NoteSmith.exe` out of its folder
- Try right-click → "Run as Administrator"

**Windows Security Warning?**
- Click "More info" → "Run anyway"
- This is normal for apps without a code signing certificate

**Forgot your password?**
- Unfortunately, passwords cannot be recovered (no feature for that yet)
- You can delete `C:\Users\YourName\.notesmith\` to start fresh
- (This will delete all your notes!)

**Need to uninstall?**
- Simply delete the NoteSmith folder
- Optionally delete `C:\Users\YourName\.notesmith\` to remove your data
